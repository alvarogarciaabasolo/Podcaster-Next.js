# Podcaster

A micro application for Music Podcasts

## Getting Started

Run dev server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

Run tests:

```bash
npm run test
```

Run build:

```bash
npm run build
```

serve build:

```bash
npm run start
```

Format code:

```bash
npm run format
```
